
### 2️⃣ **Quick Setup Commands:**

```bash
# Install dependencies
npm install

# Setup environment
npm run setup

# Configure MySQL in .env file
# Then run:
mysql -u root -p < server/database/init.sql

# Seed data
npm run seed

# Start server
npm run dev

# Open browser
# Visit: http://localhost:5000
```
