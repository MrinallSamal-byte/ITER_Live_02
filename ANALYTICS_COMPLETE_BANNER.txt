┌───────────────────────────────────────────────────────────────────────────────┐
│                        🔌 API INTEGRATIONS                                    │
└───────────────────────────────────────────────────────────────────────────────┘

✅ /api/attendance/student/:id    - Get attendance data
✅ /api/marks/student/:id         - Get marks and grades
✅ /api/events                    - Get all events
✅ /api/clubs                     - Get all clubs
✅ /api/hostel/menu              - Get hostel menu
✅ /api/timetable                - Get schedule
✅ /api/assignments/student       - Get assignments

All endpoints include automatic fallback to sample data if APIs fail!

┌───────────────────────────────────────────────────────────────────────────────┐
│                        🚀 HOW TO USE                                          │
└───────────────────────────────────────────────────────────────────────────────┘

1. Navigate to: http://localhost:5000/dashboard/student.html
2. Login with student credentials
3. Dashboard automatically loads all analytics
4. View comprehensive overview in one place
5. Scroll to see different sections:
   - Hero stats at top
   - Quick stats below hero
   - Charts and tables in middle
   - Events and clubs at bottom
   - Announcements and timeline

MANUAL REFRESH (if needed):
  window.StudentAnalytics.refresh();

ACCESS DATA (for debugging):
  console.log(window.StudentAnalytics.data);
