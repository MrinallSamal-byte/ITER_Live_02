## 🚀 How to Get Started

### 1. Quick Start (5 Minutes)
```powershell
# Install dependencies
npm install

# Configure .env file (see DEPLOYMENT_GUIDE.md)

# Run migrations
npm run migrate:security
npm run migrate:performance

# Enable event scheduler in MySQL
# SET GLOBAL event_scheduler = ON;

# Start server
npm run dev
```

### 2. Verify Installation
Visit these URLs:
- **Main App:** http://localhost:5000
- **Health Check:** http://localhost:5000/api/health
- **Detailed Metrics:** http://localhost:5000/api/health/detailed

### 3. Test Key Features
1. ✅ Login and check notification bell (top right)
2. ✅ View dashboard with charts
3. ✅ Try global search (top bar)
4. ✅ Check attendance heatmap
